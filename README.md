### Prerequisites

- Java 21 or higher
- No need to install Gradle (uses Gradle wrapper)

### Building the Project

```bash
# Build the project
./gradlew build

# Run tests
./gradlew test

# Run tests with detailed output
./gradlew test --info

# Generate test report
./gradlew test --continue
# Report available at: build/reports/tests/test/index.html
```
